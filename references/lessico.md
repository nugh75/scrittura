# Lexicon reference — Italian & English

Working material for suggestions. Not rules: flag, propose alternatives, let
the user choose.

## ITALIAN

### English calques to flag

| Calque | Why it's wrong | Better |
|---|---|---|
| realizzare (= accorgersi) | "realize" ≠ realizzare (costruire, attuare) | rendersi conto, accorgersi, capire |
| eventualmente (= alla fine) | "eventually" ≠ eventualmente (forse) | alla fine, prima o poi, col tempo |
| attualmente (= in realtà) | "actually" ≠ attualmente (adesso) | in realtà, in effetti, a dire il vero |
| supportare | anglicismo da "support" | sostenere, appoggiare, reggere |
| assumere (= supporre) | "assume" ≠ assumere (personale) | supporre, ipotizzare, presumere |
| confidente (= fiducioso) | "confident" ≠ confidente (persona fidata) | fiducioso, sicuro di sé |
| consistente (= coerente) | "consistent" ≠ consistente (solido, corposo) | coerente, uniforme, costante |
| processare (= elaborare) | "process" ≠ processare (in tribunale) | elaborare, trattare, gestire |
| applicare (per un lavoro) | "apply" ≠ applicare | candidarsi, fare domanda |
| performante | anglicismo | efficiente, ad alte prestazioni |
| implementare (uso generico) | tecnicismo abusato fuori dall'informatica | attuare, realizzare, introdurre |
| focalizzarsi su | calco di "focus on" | concentrarsi su, puntare su |
| a livello di | riempitivo da "at the level of" | riguardo a, quanto a — o riscrivere |

### Weak verbs → stronger alternatives

| Weak | Alternatives (context-dependent) |
|---|---|
| fare | svolgere, compiere, eseguire, effettuare, redigere, condurre, elaborare |
| dire | affermare, sostenere, osservare, precisare, ribadire, obiettare, ammettere |
| mettere | collocare, inserire, disporre, introdurre, apporre, riporre |
| dare | fornire, offrire, attribuire, assegnare, conferire, concedere |
| avere | possedere, presentare, disporre di, godere di, mostrare |
| cosa | aspetto, elemento, fattore, questione, fenomeno, dettaglio |
| molto + agg. | cercare l'aggettivo forte: molto grande → enorme; molto bello → splendido; molto stanco → esausto |

### Connectives by function

- **Addition**: inoltre, in aggiunta, per di più, altresì (formal), nonché
- **Contrast**: tuttavia, però, eppure, ciononostante, al contrario, d'altro canto, sebbene, benché
- **Cause**: poiché, dato che, dal momento che, in quanto, giacché (formal)
- **Consequence**: quindi, pertanto, di conseguenza, perciò, dunque, sicché
- **Concession**: pur, nonostante, malgrado, per quanto, ammesso che
- **Exemplification**: ad esempio, per esempio, in particolare, nello specifico, si pensi a
- **Reformulation**: cioè, ossia, vale a dire, in altre parole, detto altrimenti
- **Conclusion**: in conclusione, in sintesi, in definitiva, insomma, in ultima analisi

Watch for overuse: "inoltre" and "quindi" repeated in every paragraph — vary
or restructure.

### Fillers and bureaucratese to flag

| Filler | Better |
|---|---|
| al fine di | per |
| in data odierna | oggi |
| nella misura in cui | se, quando |
| a fronte di | davanti a, rispetto a — often deletable |
| in relazione a | su, riguardo a |
| provvedere a + inf. | just the verb: "provvedere a inviare" → "inviare" |
| effettuare un controllo | controllare |
| porre in essere | fare, attuare |
| il suddetto / di cui sopra | repeat the noun or use a pronoun |

### Common traps

- **D eufonica**: keep only between identical vowels ("ed era", "ad Ancona");
  "ad esempio" is a lexicalized exception. Flag "ed anche", "od altro".
- **Piuttosto che** with disjunctive meaning ("o") is wrong: piuttosto che =
  "instead of", never "or".
- **Gerund chains**: more than one gerund per sentence → rewrite with
  subordinate clauses.
- **Nominal style overuse**: "la valutazione dell'implementazione della
  soluzione" → prefer verbs: "valutare come è stata implementata la soluzione".

## ENGLISH

### False friends (Italian → English)

| Italian writes | Means in English | Intended word |
|---|---|---|
| argument (da "argomento") | litigio, ragionamento | topic, subject |
| actually (da "attualmente") | in realtà | currently, at present |
| sensible (da "sensibile") | sensato | sensitive |
| sympathetic (da "simpatico") | comprensivo | nice, likeable |
| convenient (da "conveniente") | comodo | cheap, good value / advantageous |
| eventually (da "eventualmente") | alla fine | possibly, if necessary |
| library (da "libreria") | biblioteca | bookshop |
| factory (da "fattoria") | fabbrica | farm |
| parents (da "parenti") | genitori | relatives |
| morbid (da "morbido") | morboso | soft |
| annoyed (da "annoiato") | infastidito | bored |
| comprehensive (da "comprensivo") | esaustivo | understanding |
| pretend (da "pretendere") | fingere | expect, demand |
| assist (da "assistere a") | aiutare | attend, witness |

### Italian interference patterns

- **Comma splice**: "The method is fast, it uses caching." → split with a
  period, or use "because/so/and", or a semicolon.
- **Preposition transfer**: "depends from" → depends on; "participate to" →
  participate in; "discuss about" → discuss; "married with" → married to.
- **Missing/extra articles**: "the nature is beautiful" → "nature is
  beautiful"; "I am engineer" → "I am an engineer".
- **Adjective after noun**: "a solution elegant" → "an elegant solution".
- **Present for present perfect**: "I live here since 2010" → "I have lived
  here since 2010".
- **Long periodic sentences**: Italian tolerates 40+ word sentences with
  nested clauses; English prefers shorter units. Suggest splitting.
- **"In fact" as filler**: Italian "infatti" (confirmation) ≠ English "in
  fact" (contrast/correction). Often deletable or → "indeed".

### Weak verbs → stronger alternatives

| Weak | Alternatives (context-dependent) |
|---|---|
| do | perform, carry out, conduct, execute, complete |
| make | create, produce, build, generate, form, cause |
| get | obtain, receive, acquire, become, reach, retrieve |
| say | state, claim, argue, note, point out, acknowledge, contend |
| put | place, insert, set, position, apply |
| very + adj. | find the strong adjective: very big → huge; very important → crucial; very tired → exhausted |
| thing | aspect, factor, element, issue, feature, point |

### Transitions by function

- **Addition**: moreover, furthermore, in addition, also, besides
- **Contrast**: however, yet, nevertheless, nonetheless, on the other hand, whereas, although
- **Cause**: because, since, as, given that, owing to
- **Consequence**: therefore, thus, consequently, as a result, hence
- **Exemplification**: for example, for instance, in particular, notably, such as
- **Reformulation**: that is, in other words, put differently, namely
- **Conclusion**: in conclusion, in short, overall, ultimately, to sum up

Watch for overuse: "however" and "moreover" opening every paragraph — vary or
restructure.

### Wordiness to flag

| Wordy | Better |
|---|---|
| in order to | to |
| due to the fact that | because |
| at this point in time | now |
| in the event that | if |
| a large number of | many |
| it is important to note that | (delete, or "note that") |
| has the ability to | can |
| in spite of the fact that | although |
| with regard to | about, regarding |
| conduct an analysis of | analyze |
